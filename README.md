# calculator

## About
Small calculator project based on rest api.
## Quickstart

Build docker image:

    $ docker build -t calculator .

Run image:

    $ docker run -p 8000:8000 calculator

Checkout api endpoints at:
    
    http://127.0.0.1:8000/api/v1/schema/swagger-ui/

Play with calculator at:

    http://127.0.0.1:8000/
